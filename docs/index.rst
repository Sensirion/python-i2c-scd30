Sensirion SCD30 I2C Python Driver
=================================

This package contains the driver for the Sensirion I2C SCD30 sensor.
It provides a hardware-independent interface which works with various I2C transceivers.

Click [here](https://sensirion.com/products/catalog/SCD30/) to learn more about the Sensirion SCD30 sensor.
